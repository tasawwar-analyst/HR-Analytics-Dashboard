# HR Analytics Dashboard

## Project Overview

This project is an interactive HR Analytics Dashboard developed using Power BI.
The dashboard provides detailed insights into employee attendance, work from home trends, sick leave analysis, and workforce presence patterns.

The main objective of this project is to help HR teams monitor employee performance, analyze attendance behavior, track leave trends,
and support data-driven workforce management decisions using interactive visualizations and KPI tracking.

--------------------------------------------------

## Business Problem

Organizations generate large amounts of employee attendance and workforce data daily. Without proper analysis, it becomes difficult to:

* Monitor employee presence performance  
* Track work from home trends  
* Analyze sick leave patterns  
* Identify attendance fluctuations  
* Compare employee attendance behavior  
* Understand workforce productivity trends  
* Improve HR decision-making processes  

This dashboard solves these challenges through interactive reporting and data visualization.

--------------------------------------------------

## Tools & Technologies Used

* Power BI — Dashboard Development  
* Power Query — Data Cleaning & Transformation  
* DAX — KPI Calculations  
* Excel / CSV — Dataset Management  
* GitHub — Project Hosting  

--------------------------------------------------

## Dashboard Features

### Presence Insights Dashboard

The dashboard provides a complete overview of employee attendance and workforce performance including:

* Employee Presence Percentage  
* Work From Home (WFH) Analysis  
* Sick Leave (SL) Analysis  
* Presence Trends by Date  
* WFH Trends by Date  
* SL Trends by Date  
* Employee-wise Attendance Tracking  
* Day-wise Attendance Performance  
* Attendance Status Monitoring  
* Interactive Filters & Monthly Analysis  

--------------------------------------------------

## KPI Metrics Used

* Presence % — Overall employee attendance percentage  
* WFH % — Work from home percentage  
* SL % — Sick leave percentage  
* Employee Attendance — Individual employee attendance tracking  
* Day-wise Performance — Attendance analysis by weekdays  

--------------------------------------------------

## Key Insights

* Overall employee presence remained above 90%  
* Work from home percentage averaged around 10%  
* Sick leave percentage remained low throughout the analysis period  
* Monday showed the highest employee presence percentage  
* Friday recorded the highest work from home percentage  
* Attendance patterns fluctuated during certain periods indicating workforce variation  
* Most employees maintained consistent attendance performance
